**POINTERS**
int a=10;
int *ap=&a;
- &: address
- *: Dereferencing (value)
- Symbol table: Map of address and variables
- int: 4 bytes
- pointer to int: 8 bytes(not fixed)
  
  (*p)++;  // Increments the value of var held by address stored in p
